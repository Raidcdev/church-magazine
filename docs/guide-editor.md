# 교정자 사용설명서 📝

> 교회 계간지 원고 제출 시스템 — 교정자(교정하는 분)용 안내서

---

## 1. 로그인하기

1. 인터넷 브라우저에서 사이트 주소를 입력합니다
2. **이름**과 **비밀번호(숫자 4자리)** 를 입력합니다
3. **로그인** 버튼을 누릅니다

> 로그인하면 자동으로 "교정 작업" 화면으로 이동합니다.

---

## 2. 교정 작업 화면 살펴보기

### 화면 위쪽: 제작 일정

```
원고 마감          인쇄소 전달        출간 목표
2025.03.15        2025.04.01        2025.04.15
  D-14              D-31              D-45
```

- 일정은 **읽기만** 가능합니다 (수정은 관리자만 가능)

### 화면 중간: 전체 진행도

```
전체 진행: 3/17 확정  ▓▓▓░░░░░░░░░░░░░░
```

- "펼쳐보기"를 누르면 전체 목차별 상태를 한눈에 볼 수 있습니다

### 필터 탭 (중요!)

목록 위에 필터 버튼이 있습니다:

```
[ 전체(17) ] [ 미배정(2) ] [ 미제출(5) ] [ 제출완료(4) ] [ 교정중(3) ] [ 교정완료(2) ] [ 확정(1) ]
```

- **제출완료**: 필자가 제출하고 교정을 기다리는 원고들
- **교정중**: 내가 교정 작업을 시작한 원고들
- **필터를 눌러서** 원하는 상태의 원고만 골라 볼 수 있습니다

## 3. 원고 목록 보기

### 컴퓨터 화면 (테이블 형태)

```
┌────┬────────┬──────────────────┬────────┬───────────┬─────┬────────┐
│ #  │ 목차   │ 제목             │ 필자   │ 상태      │파일 │        │
├────┼────────┼──────────────────┼────────┼───────────┼─────┼────────┤
│ 1  │ 1      │ 새물결 뉴우스     │ 김철수 │ 🔵제출완료│ 2   │교정가능│
│ 2  │ 2      │ 내일을 향한 기도  │ 이영희 │ 🟡교정중  │ 0   │교정가능│
│ 3  │ 3-1    │ 축하 인사        │ 미배정 │ ⚪미제출   │ 0   │        │
└────┴────────┴──────────────────┴────────┴───────────┴─────┴────────┘
```

### 휴대폰 화면 (카드 형태)

```
┌─────────────────────────────┐
│ [1] 새물결 뉴우스            │
│ 김철수 · 🔵 제출완료 · 📎 2  │
│                    교정가능  │
└─────────────────────────────┘
```

- **"교정가능"** 이라고 표시된 항목만 교정할 수 있습니다
- 항목을 **누르면** 상세 화면이 뜹니다

---

## 4. 필자 배정하기

필자가 아직 정해지지 않은 항목(**"미배정"**)에 필자를 배정할 수 있습니다.

1. **"미배정"** 이라고 적힌 보라색 버튼을 누릅니다
2. 필자 목록이 드롭다운으로 나옵니다
3. 배정할 필자 이름을 선택합니다
4. 즉시 배정됩니다

```
미배정  →  눌러서 선택  →  [홍길동 ▼]  →  배정 완료!
```

---

## 5. 항목(목차) 추가하기

새로운 원고 항목을 추가할 수 있습니다.

1. 화면 맨 아래의 **"+ 항목 추가"** 버튼을 누릅니다
2. 팝업이 뜹니다:

```
┌─────────────────────────────┐
│ 항목 추가                    │
│                             │
│ 순번: [18     ]             │
│ 목차 번호: [12   ]          │
│ 제목: [새 원고 제목  ]       │
│ 필자: [홍길동 선택 ▼]       │
│                             │
│    [ 추가하기 ]              │
└─────────────────────────────┘
```

3. **순번**: 목록에서 보여질 순서 (숫자)
4. **목차 번호**: 인쇄본에 들어갈 번호 (예: 12, 7-1)
5. **제목**: 원고 제목
6. **필자**: 배정할 사람 선택 (나중에 정해도 됨)
7. **"추가하기"** 를 누르면 목록에 추가됩니다

## 6. 원고 상세 보기

목록에서 항목을 누르면 **상세 팝업**이 뜹니다.

```
┌─────────────────────────────────────────────┐
│ 🔵 제출완료                    📎 파일 2개   │
│                                             │
│ 새물결 뉴우스                                │
│ 필자: 김철수                                 │
│ 제출: 2025.03.10 14:30                      │
│                                             │
│ ┌───────────────┬───────────────┐           │
│ │   원본 본문    │  교정 본문     │           │
│ │               │               │           │
│ │ 필자가 쓴     │ (교정 전이면   │           │
│ │ 원래 글       │  비어있음)     │           │
│ └───────────────┴───────────────┘           │
│                                             │
│ [교정하기]  [정보 수정]  [닫기]               │
└─────────────────────────────────────────────┘
```

- **원본 본문**: 필자가 제출한 글 (왼쪽)
- **교정 본문**: 교정된 글 (오른쪽)
  - 교정이 완료되면 **변경 부분이 색깔로 표시**됩니다
  - 🔴 빨간 취소선: 삭제된 부분
  - 🟢 초록 배경: 추가된 부분

### 버튼 설명

| 버튼 | 언제 보이나 | 기능 |
|------|-----------|------|
| **교정하기** | 🔵제출완료 또는 🟡교정중 | 교정 화면으로 이동 |
| **정보 수정** | 항상 | 제목, 목차번호, 필자 수정 |
| **삭제** | ⚪미제출인 경우만 | 항목 삭제 |
| **닫기** | 항상 | 팝업 닫기 |

## 7. 교정하기 (가장 중요!)

### 7-1. 교정 화면 열기

1. 상세 팝업에서 **"교정하기"** 버튼을 누릅니다
2. 교정 화면이 열립니다

### 7-2. 교정 화면 구성

**컴퓨터 화면** (좌우 분할):

```
┌─────────────────────┬─────────────────────┐
│    📖 원본           │    ✏️ 교정           │
│                     │                     │
│  필자가 쓴 원래 글   │  여기를 수정합니다    │
│  (수정 불가)         │  (자유롭게 편집)     │
│                     │                     │
│                     │                     │
│  📷 첨부 사진들      │                     │
│  📎 참고파일 목록    │                     │
│               245자  │               252자  │
└─────────────────────┴─────────────────────┘
│                                           │
│    [ 임시저장 ]        [ 교정완료 ]         │
└───────────────────────────────────────────┘
```

**휴대폰 화면** (탭 전환):

```
[ 원본 ]  [ 교정 ]    ← 탭을 눌러 전환

┌─────────────────────┐
│                     │
│  (선택한 탭 내용)    │
│                     │
└─────────────────────┘

[ 임시저장 ]  [ 교정완료 ]
```

### 7-3. 교정 작업하기

1. **왼쪽(원본)** 을 보면서 **오른쪽(교정)** 에 수정합니다
   - 처음 교정을 시작하면 원본 내용이 교정란에 복사되어 있습니다
   - 필요한 부분을 수정하면 됩니다
2. 왼쪽 아래에 **첨부된 사진**과 **참고파일**이 있으니 참고하세요
   - 사진: 누르면 크게 볼 수 있습니다
   - 참고파일: 누르면 다운로드/열기가 됩니다
3. 양쪽의 **글자 수**를 비교할 수 있습니다

### 7-4. 임시저장

- 교정 중간에 저장하고 싶으면 **"임시저장"** 을 누릅니다
- 상태가 **🟡 교정중**으로 바뀝니다
- 나중에 다시 와서 이어서 교정할 수 있습니다

> 💡 필자가 교정중인 원고를 수정하지 못하도록 잠김 상태가 됩니다.

### 7-5. 교정완료

교정을 다 마쳤으면:

1. **"교정완료"** 버튼을 누릅니다
2. "교정을 완료하시겠습니까?" 확인 메시지가 뜹니다
3. **"확인"** 을 누르면 상태가 **🟣 교정완료**로 바뀝니다
4. 관리자가 확인하고 최종 확정합니다

> ⚠️ 교정완료 후에는 수정이 안 됩니다. 신중하게 확인한 후 완료하세요.

> ⚠️ 교정 내용이 비어있으면 교정완료 버튼이 눌리지 않습니다.

---

## 8. 정보 수정하기

항목의 기본 정보를 수정할 수 있습니다.

1. 상세 팝업에서 **"정보 수정"** 을 누릅니다
2. 수정 팝업이 뜹니다:

```
┌─────────────────────────────┐
│ 정보 수정                    │
│                             │
│ 순번: [1      ]             │
│ 목차 번호: [3-1  ]          │
│ 제목: [축하 인사    ]        │
│ 필자: [이정철 목사님 ▼]      │
│                             │
│    [ 수정 완료 ]             │
└─────────────────────────────┘
```

3. 수정 후 **"수정 완료"** 를 누르면 반영됩니다

## 9. 항목 삭제하기

- **⚪ 미제출(draft)** 상태인 항목만 삭제할 수 있습니다
- 필자가 이미 제출한 항목은 삭제할 수 없습니다

1. 상세 팝업에서 **"삭제"** 버튼을 누릅니다 (미제출인 경우만 표시)
2. 확인 메시지가 뜨면 **"확인"** 을 누릅니다

> ⚠️ 삭제하면 되돌릴 수 없습니다. 신중하게 결정하세요.

## 교정 작업 순서 한눈에 보기

```
① 로그인
    ↓
② 필터에서 "제출완료" 선택 → 교정할 원고 찾기
    ↓
③ 항목 클릭 → 상세 팝업 → "교정하기" 클릭
    ↓
④ 원본을 보면서 교정본 수정
    ↓
⑤ 중간 저장은 "임시저장"
    ↓
⑥ 다 되면 "교정완료"
    ↓
⑦ 관리자가 최종 확정
    ↓
완료! 🎉
```

## 상태별 교정자가 할 수 있는 것

| 상태 | 할 수 있는 것 |
|------|-------------|
| ⚪ 미제출 | 정보 수정, 삭제 |
| 🔵 제출완료 | **교정하기**, 정보 수정 |
| 🟡 교정중 | **교정 이어하기**, 정보 수정 |
| 🟣 교정완료 | 정보 수정 (교정 본문 수정 불가) |
| 🟢 확정 | 정보 수정 (교정 본문 수정 불가) |

## 자주 묻는 질문 (FAQ)

### Q. 교정완료를 눌렀는데 다시 수정하고 싶어요
→ 교정완료 후에는 수정할 수 없습니다. 관리자에게 연락하세요.

### Q. 필자가 글을 수정했다고 하는데 제출완료가 안 보여요
→ 필자가 "임시 저장"만 하고 "제출하기"를 안 눌렀을 수 있습니다. 필자에게 "제출하기"를 눌러달라고 안내하세요.

### Q. 비밀번호를 잊어버렸어요
→ 관리자에게 연락하면 비밀번호를 새로 설정해 줍니다.

### Q. 첨부파일이 안 열려요
→ 파일 종류에 따라 해당 프로그램이 필요합니다 (예: .hwp는 한글, .docx는 워드).
