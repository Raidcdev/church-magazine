# 필자 사용설명서 ✏️

> 교회 계간지 원고 제출 시스템 — 필자(글쓰는 분)용 안내서

---

## 1. 로그인하기

### 첫 화면에서 로그인합니다

1. 인터넷 브라우저(크롬, 사파리 등)에서 **사이트 주소**를 입력합니다
2. 아래와 같은 로그인 화면이 나옵니다

```
┌─────────────────────────┐
│      ✚ 교회 계간지       │
│     원고 제출 시스템      │
│                         │
│  이름: [홍길동        ]  │
│  비밀번호: [● ● ● ●  ]  │
│                         │
│      [ 로그인 ]          │
└─────────────────────────┘
```

3. **이름**: 관리자가 알려준 본인 이름을 입력합니다 (예: 홍길동)
4. **비밀번호**: 숫자 4자리를 입력합니다 (예: 1234)
5. **로그인** 버튼을 누릅니다

> 💡 비밀번호를 모르겠으면 관리자에게 문의하세요.

---

## 2. 내 화면 살펴보기

로그인하면 **내 원고 목록**이 나옵니다.

### 화면 위쪽: 제작 일정

```
원고 마감          인쇄소 전달        출간 목표
2025.03.15        2025.04.01        2025.04.15
  D-14              D-31              D-45
```

- **D-숫자**: 그 날짜까지 남은 일수입니다
- 빨간색이면 3일 이내, 주황색이면 7일 이내입니다
- 마감일을 꼭 확인하세요!

### 화면 중간: 전체 진행도

```
전체 진행: 3/17 확정  ▓▓▓░░░░░░░░░░░░░░
```

- 전체 계간지 원고가 몇 편이나 완료되었는지 보여줍니다
- **"펼쳐보기"** 를 누르면 전체 목차를 볼 수 있습니다
- 내 원고는 **파란색으로 표시**되어 쉽게 찾을 수 있습니다

### 화면 아래쪽: 내 원고 카드

배정된 원고가 카드 형태로 나옵니다.

```
┌─────────────────────────────────────┐
│  [3-1]  창간호 축하 인사             │
│         ⚪ 미제출                    │
│                                     │
│  작성 ─── 제출 ─── 교정 ─── 확정    │
│   ●       ○       ○       ○        │
└─────────────────────────────────────┘
```

- 카드를 **누르면** 펼쳐지면서 글을 쓸 수 있는 공간이 나옵니다
- 상태 색깔 의미:
  - ⚪ **미제출** (회색): 아직 글을 제출하지 않은 상태
  - 🔵 **제출완료** (파란색): 제출 완료, 교정 대기 중
  - 🟡 **교정중** (노란색): 교정자가 교정하고 있음
  - 🟣 **교정완료** (보라색): 교정이 끝남
  - 🟢 **확정** (초록색): 관리자가 최종 확정함

## 3. 원고 쓰기 (가장 중요!)

### 3-1. 카드를 눌러 펼칩니다

카드를 누르면 아래처럼 글쓰기 공간이 나옵니다:

```
┌─────────────────────────────────────┐
│  [3-1]  창간호 축하 인사             │
│                                     │
│  ┌─────────────────────────────┐    │
│  │                             │    │
│  │  (여기에 글을 씁니다)        │    │
│  │                             │    │
│  │                             │    │
│  │                    247자 ───┘    │
│  └─────────────────────────────┘    │
│                                     │
│  📷 사진 추가    📎 참고파일 첨부     │
│                                     │
│  [ 임시 저장 ]    [ 제출하기 ]       │
└─────────────────────────────────────┘
```

### 3-2. 글을 씁니다

- 빈 공간을 누르고 글을 작성합니다
- 오른쪽 아래에 **글자 수**가 자동으로 표시됩니다
- 길이 제한은 없으니 편하게 쓰세요

### 3-3. 사진 첨부하기 (필요한 경우)

1. **"📷 사진 추가"** 버튼을 누릅니다
2. 휴대폰이나 컴퓨터에서 사진 파일을 선택합니다
3. 선택한 사진이 작은 네모(썸네일)로 표시됩니다
4. 사진을 누르면 크게 볼 수 있습니다
5. 잘못 올렸으면 사진 옆의 **X** 를 눌러 삭제합니다

### 3-4. 참고파일 첨부하기 (필요한 경우)

1. **"📎 참고파일 첨부"** 버튼을 누릅니다
2. 한글(.hwp), 워드(.doc), PDF 등의 파일을 선택합니다
3. 파일 이름이 목록에 표시됩니다
4. 잘못 올렸으면 옆의 **X** 를 눌러 삭제합니다

> 📌 올릴 수 있는 파일 종류: 한글(.hwp), 워드(.doc, .docx), PDF, 텍스트(.txt), 엑셀(.xlsx), 파워포인트(.pptx), 압축(.zip)

## 4. 임시 저장과 제출

### "임시 저장" 과 "제출하기" 의 차이

| 구분 | 임시 저장 | 제출하기 |
|------|----------|---------|
| 의미 | 작성 중인 글을 저장만 함 | 교정자에게 글을 보냄 |
| 상태 | **미제출** 유지 | **제출완료**로 변경 |
| 수정 가능? | ✅ 언제든 수정 가능 | ✅ 교정 시작 전까지 수정 가능 |
| 여러 번 가능? | ✅ 몇 번이든 가능 | ✅ "다시 제출" 가능 |

### 임시 저장하기

1. 글을 쓰다가 **"임시 저장"** 버튼을 누릅니다
2. "저장되었습니다" 메시지가 나옵니다
3. 나중에 다시 와서 이어 쓸 수 있습니다

> 💡 글을 쓰다가 중간에 저장하고 싶을 때 사용하세요. 교정자에게는 아직 보이지 않습니다.

### 제출하기

1. 글을 다 쓰셨으면 **"제출하기"** 버튼을 누릅니다
2. 확인 메시지가 나오면 **"확인"** 을 누릅니다
3. 상태가 **🔵 제출완료**로 바뀝니다
4. 교정자가 이제 내 글을 볼 수 있습니다

> ⚠️ 글 내용이 비어있으면 제출 버튼이 눌리지 않습니다. 꼭 내용을 쓴 후에 제출하세요.

### 제출 후 수정이 필요하면?

제출했는데 수정하고 싶다면:

1. 카드를 눌러 펼칩니다
2. 글을 수정합니다
3. **"임시 저장"** 을 누르면 → 상태가 **⚪ 미제출**로 돌아갑니다 (제출 취소됨)
4. 다시 수정 후 **"다시 제출"** 을 누르면 됩니다

> ⚠️ **교정자가 이미 교정을 시작했으면** (🟡 교정중) 수정할 수 없습니다!
> 이 경우 교정자에게 직접 연락하세요.

## 5. 상태별 내가 할 수 있는 것

| 상태 | 내가 할 수 있는 것 |
|------|------------------|
| ⚪ 미제출 | 글쓰기, 수정, 사진/파일 첨부, 임시저장, 제출 |
| 🔵 제출완료 | 글 수정, 임시저장(→미제출로 돌아감), 다시 제출 |
| 🟡 교정중 | ❌ 수정 불가 (읽기만 가능) |
| 🟣 교정완료 | ❌ 수정 불가 (읽기만 가능) |
| 🟢 확정 | ❌ 수정 불가 (읽기만 가능) |

> 교정이 시작되면(🟡) 부터는 글을 수정할 수 없습니다.
> **마감일 전에 꼭 제출해 주세요!**

## 자주 묻는 질문 (FAQ)

### Q. 비밀번호를 잊어버렸어요
→ 관리자에게 연락하면 비밀번호를 새로 설정해 줍니다.

### Q. 글을 제출했는데 다시 수정하고 싶어요
→ 상태가 🔵 제출완료이면 "임시 저장"을 눌러 제출을 취소하고, 수정 후 다시 제출하세요.

### Q. 교정중이라고 나오는데 수정이 안 돼요
→ 교정자가 이미 교정을 시작한 것입니다. 수정이 필요하면 교정자에게 직접 말씀해 주세요.

### Q. 사진이 안 올라가요
→ 사진 파일(jpg, png 등)만 올릴 수 있습니다. 파일 크기가 너무 크면 안 될 수 있습니다.

### Q. 내 원고가 안 보여요
→ 관리자가 아직 배정하지 않은 것입니다. 관리자에게 문의하세요.

## 한눈에 보는 사용 순서

```
① 로그인
    ↓
② 내 카드를 눌러 펼치기
    ↓
③ 글쓰기 + 사진/파일 첨부
    ↓
④ 중간 저장은 "임시 저장"
    ↓
⑤ 다 쓰면 "제출하기"
    ↓
⑥ 교정자가 교정 → 관리자가 확정
    ↓
완료! 🎉
```
