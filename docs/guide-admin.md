# 관리자 사용설명서 🔧

> 교회 계간지 원고 제출 시스템 — 관리자용 안내서

---

## 1. 로그인하기

1. 인터넷 브라우저에서 사이트 주소를 입력합니다
2. **이름**과 **비밀번호(숫자 4자리)** 를 입력합니다
3. **로그인** 버튼을 누릅니다

> 로그인하면 자동으로 "관리자" 대시보드로 이동합니다.

---

## 2. 관리자 화면 살펴보기

### 화면 상단: 헤더

```
┌──────────────────────────────────────────────┐
│  관리자             홍길동  [회원관리] [로그아웃]│
└──────────────────────────────────────────────┘
```

- **회원관리**: 회원 추가, 삭제, 비밀번호 변경 등을 할 수 있는 페이지로 이동
- **로그아웃**: 로그아웃

### 제작 일정 (수정 가능!)

```
원고 마감          인쇄소 전달        출간 목표        [수정]
2025.03.15        2025.04.01        2025.04.15
  D-14              D-31              D-45
```

- 관리자만 **"수정"** 버튼이 보입니다
- 눌러서 날짜를 변경할 수 있습니다 (자세한 내용은 아래 참고)

### 전체 진행도

```
전체 진행: 3/17 확정  ▓▓▓░░░░░░░░░░░░░░
```

- 전체 원고 중 몇 편이 확정되었는지 한눈에 보여줍니다
- "펼쳐보기"를 누르면 전체 목차별 상태를 볼 수 있습니다

### 필터 탭

```
[ 전체(17) ] [ 미배정(2) ] [ 미제출(5) ] [ 제출완료(4) ] [ 교정중(3) ] [ 교정완료(2) ] [ 확정(1) ]
```

- 필터를 눌러서 원하는 상태의 원고만 볼 수 있습니다
- **교정완료**: 교정자가 교정을 마쳐서 관리자가 확정해야 하는 항목들

---

## 3. 원고 목록 보기

### 컴퓨터 화면

```
┌────┬────────┬──────────────────┬────────┬───────────┬─────┬───────────┐
│ #  │ 목차   │ 제목             │ 필자   │ 상태      │파일 │ 액션       │
├────┼────────┼──────────────────┼────────┼───────────┼─────┼───────────┤
│ 1  │ 1      │ 새물결 뉴우스     │ 김철수 │ 🟣교정완료│ 2   │ [확정하기] │
│ 2  │ 2      │ 내일을 향한 기도  │ 이영희 │ 🟢확정    │ 0   │ 📥 확정해제│
│ 3  │ 3-1    │ 축하 인사        │ 미배정 │ ⚪미제출   │ 0   │           │
└────┴────────┴──────────────────┴────────┴───────────┴─────┴───────────┘
```

- **확정하기** (초록 버튼): 교정완료된 원고를 최종 확정
- **📥 다운로드**: 확정된 원고를 파일로 받기
- **확정해제** (빨간 글자): 확정을 취소하고 교정완료 상태로 되돌리기

---

## 4. 원고 확정하기 (관리자의 핵심 업무!)

교정자가 교정을 완료하면(**🟣 교정완료**) 관리자가 확인하고 확정합니다.

### 목록에서 바로 확정하기

1. 🟣 교정완료 상태인 항목의 **"확정하기"** 버튼을 누릅니다
2. 확인 메시지가 뜨면 **"확인"** 을 누릅니다
3. 상태가 **🟢 확정**으로 바뀝니다

### 상세 화면에서 확정하기 (원본/교정본 비교 후)

1. 항목을 **클릭**하여 상세 팝업을 엽니다
2. **원본(왼쪽)** 과 **교정본(오른쪽)** 을 비교합니다
   - 🔴 빨간 취소선: 삭제된 부분
   - 🟢 초록 배경: 추가된 부분
3. 교정 내용이 적절하면 **"확정하기"** 버튼을 누릅니다

```
┌─────────────────────┬─────────────────────┐
│    📖 원본           │    ✏️ 교정본         │
│                     │                     │
│  창간호를 맞이하여   │  창간호를 맞이하여   │
│  정말 기쁩니다.      │  매우 기쁩니다.      │  ← 변경 표시
│                     │                     │
└─────────────────────┴─────────────────────┘

[ 정보 수정 ]  [ 확정하기 ]  [ 닫기 ]
```

> 💡 교정 내용을 꼼꼼히 확인한 후에 확정하세요. 확정 후에도 해제할 수 있지만, 확인 습관을 들이는 것이 좋습니다.

---

## 5. 확정된 원고 다운로드

확정(**🟢**)된 원고는 다운로드할 수 있습니다 (인쇄소에 전달할 때 사용).

### 목록에서 다운로드

1. 🟢 확정 상태인 항목의 **📥 다운로드** 버튼을 누릅니다
2. 교정본이 **.txt 파일**로 다운로드됩니다
3. 첨부된 사진/파일이 있으면 새 탭에서 열립니다

### 상세 화면에서 다운로드

1. 항목을 클릭하여 상세 팝업을 엽니다
2. **"다운로드"** 버튼을 누릅니다

---

## 6. 확정해제 (되돌리기)

확정을 실수로 했거나 수정이 필요한 경우:

1. 🟢 확정 상태인 항목의 **"확정해제"** 를 누릅니다
2. 확인 메시지가 뜨면 **"확인"** 을 누릅니다
3. 상태가 **🟣 교정완료**로 되돌아갑니다

> 교정자가 다시 수정하길 원하면 확정해제 후 교정자에게 연락하세요.

---

## 7. 필자 배정하기

필자가 정해지지 않은 항목에 필자를 배정합니다.

1. **"미배정"** 이라고 적힌 보라색 버튼을 누릅니다
2. 필자 목록이 드롭다운으로 나옵니다
3. 배정할 필자를 선택합니다
4. 즉시 배정됩니다

---

## 8. 항목(목차) 추가하기

1. 화면 맨 아래 **"+ 항목 추가"** 를 누릅니다
2. 팝업에서 입력합니다:
   - **순번**: 목록 정렬 순서 (숫자)
   - **목차 번호**: 인쇄본 목차 번호 (예: 12, 7-1)
   - **제목**: 원고 제목
   - **필자**: 배정할 사람 (나중에 정해도 됨)
3. **"추가하기"** 를 누릅니다

---

## 9. 항목 정보 수정 / 삭제

### 정보 수정

1. 항목을 클릭하여 상세 팝업을 엽니다
2. **"정보 수정"** 을 누릅니다
3. 순번, 목차 번호, 제목, 필자를 수정합니다
4. **"수정 완료"** 를 누릅니다

### 항목 삭제

- **⚪ 미제출(draft)** 상태인 항목만 삭제 가능합니다
- 필자가 이미 제출한 항목은 삭제할 수 없습니다 (데이터 보호)

1. 상세 팝업에서 **"삭제"** 를 누릅니다
2. 확인 메시지가 뜨면 **"확인"** 을 누릅니다

> ⚠️ 삭제하면 되돌릴 수 없습니다.

---

## 10. 제작 일정 수정하기

1. 제작 일정 영역의 **"수정"** 버튼을 누릅니다
2. 일정 수정 팝업이 뜹니다:

```
┌─────────────────────────────┐
│ 제작 일정 수정               │
│                             │
│ 원고 마감:    [2025-03-15]  │
│ 인쇄소 전달:  [2025-04-01]  │
│ 출간 목표:    [2025-04-15]  │
│                             │
│   [ 수정 완료 ]  [ 취소 ]    │
└─────────────────────────────┘
```

3. 날짜를 변경합니다 (날짜 칸을 누르면 달력이 나옵니다)
4. **"수정 완료"** 를 누릅니다
5. 모든 사용자(필자, 교정자)의 화면에 바뀐 일정이 표시됩니다

> 💡 마감일이 변경되면 필자와 교정자에게 별도로 알려주세요 (시스템에서 자동 알림은 없습니다).

## 11. 회원 관리 (중요!)

### 회원관리 페이지로 이동

- 화면 오른쪽 위의 **"회원관리"** 버튼을 누릅니다

### 회원관리 화면 구성

```
┌──────────────────────────────────────────────┐
│  ← 대시보드          회원 관리     전체 15명   │
├──────────────────────────────────────────────┤
│  [관리자 2명]    [교정자 3명]    [필자 10명]   │
├──────────────────────────────────────────────┤
│                                              │
│  홍길동  [관리자]                              │
│         [권한변경] [비번변경]                   │
│                                              │
│  김철수  [필자]  원고배정됨                     │
│         [권한변경] [비번변경]                   │
│                                              │
│  이영희  [교정자]                              │
│         [권한변경] [비번변경] [삭제]            │
│                                              │
│  + 회원 추가                                  │
└──────────────────────────────────────────────┘
```

### 11-1. 회원 추가하기

1. **"+ 회원 추가"** 를 누릅니다
2. 입력 폼이 나옵니다:

```
이름: [박영수        ]
비밀번호: [1234]  (숫자 4자리)
역할: [필자 ▼]       (필자 / 교정자 / 관리자)

[ 추가하기 ]  [ 취소 ]
```

3. 정보를 입력하고 **"추가하기"** 를 누릅니다

> ⚠️ 이름은 중복될 수 없습니다. 이미 있는 이름이면 오류가 납니다.

### 11-2. 비밀번호 변경하기

필자나 교정자가 비밀번호를 잊어버렸을 때:

1. 해당 회원의 **"비번변경"** 을 누릅니다
2. 새 비밀번호(숫자 4자리)를 입력합니다
3. **"변경"** 을 누릅니다

```
새 비밀번호: [5678]  (숫자 4자리)
[ 변경 ]  [ 취소 ]
```

> 💡 본인의 비밀번호도 이 화면에서 바꿀 수 있습니다.

### 11-3. 권한(역할) 변경하기

필자를 교정자로, 교정자를 관리자로 바꿀 때:

1. 해당 회원의 **"권한변경"** 을 누릅니다
2. 새 역할 버튼이 나옵니다:

```
현재: [교정자] (회색, 선택 불가)
변경: [필자]  [관리자]    ← 원하는 역할 클릭

[ 취소 ]
```

3. 원하는 역할 버튼을 누르면 **즉시 변경**됩니다

> ⚠️ 본인의 권한은 변경할 수 없습니다 (실수 방지).

### 11-4. 회원 삭제하기

1. 해당 회원의 **"삭제"** 를 누릅니다
2. 확인 메시지가 뜨면 **"확인"** 을 누릅니다

> ⚠️ 삭제할 수 없는 경우:
> - **관리자** 계정은 삭제 불가
> - **원고가 배정된 필자**는 삭제 불가 ("원고배정됨" 배지가 있는 사람)
>   → 먼저 배정된 원고를 다른 사람에게 변경하거나 항목을 삭제한 후에 회원을 삭제하세요

### 11-5. 대시보드로 돌아가기

- 왼쪽 위의 **"← 대시보드"** 를 누르면 관리자 대시보드로 돌아갑니다

## 관리자 업무 순서 한눈에 보기

### 초기 세팅 (처음 한 번)

```
① 회원관리에서 필자, 교정자 계정 만들기
    ↓
② 제작 일정 설정하기 (원고 마감 / 인쇄소 전달 / 출간 목표)
    ↓
③ 목차 항목 추가하기
    ↓
④ 각 항목에 필자 배정하기
```

### 일상 관리

```
⑤ 필자들이 원고 제출할 때까지 기다림
    ↓
⑥ 교정자가 교정을 완료할 때까지 기다림
    ↓
⑦ 🟣 교정완료 항목을 원본/교정본 비교 후 확정하기
    ↓
⑧ 모든 항목이 🟢 확정되면 다운로드하여 인쇄소 전달
```

## 자주 묻는 질문 (FAQ)

### Q. 필자가 비밀번호를 잊어버렸다고 합니다
→ 회원관리 → 해당 필자 → "비번변경" → 새 비밀번호(숫자 4자리) 입력 → "변경"

### Q. 새로운 필자를 추가하고 싶어요
→ 회원관리 → "+ 회원 추가" → 이름, 비밀번호, 역할(필자) 입력 → "추가하기"

### Q. 원고를 잘못 확정했어요
→ 해당 항목의 "확정해제"를 누르면 🟣 교정완료 상태로 돌아갑니다.

### Q. 교정 내용이 마음에 안 들어요
→ 확정하지 말고 교정자에게 직접 연락하여 수정을 요청하세요. 확정해제 후 교정자가 다시 교정할 수 있습니다.

### Q. 필자를 삭제하고 싶은데 안 돼요
→ "원고배정됨" 배지가 있는 필자는 삭제할 수 없습니다. 먼저 해당 필자에게 배정된 원고를 다른 사람으로 변경하거나 항목을 삭제하세요.

### Q. 마감일을 변경했는데 필자들이 모를까 봐 걱정이에요
→ 시스템에 자동 알림 기능은 없습니다. 마감일 변경 후 단체 문자나 카톡으로 알려주세요.

### Q. 인쇄소에 전달할 파일은 어떻게 받나요?
→ 🟢 확정된 항목의 "다운로드" 버튼을 누르면 교정본이 .txt 파일로 저장됩니다. 첨부 사진은 별도로 새 탭에서 열립니다.
